📌 App Purpose

The purpose of this app is to help users maintain social connections during busy days. Many people forget to reach out to friends or family, so this app provides simple suggestions based on the time of day.

The app is designed to be quick and easy to use, allowing users to enter a time (such as morning or evening) and instantly receive a social interaction idea.

⚙️ How the App Works

The user enters a time of day into the input field. When the “Get Suggestion” button is clicked, the app uses if statements to check the input and display a suitable suggestion.

For example:

Morning → Send a good morning message Afternoon → Share a meme Night → Comment on a friend’s post

If the user enters invalid input, the app displays an error message to guide them.

The reset button clears both the input field and the result.

🎨 Design Considerations

The app was designed with simplicity and usability in mind:

A clean and modern interface was used A background image with overlay improves visual appeal Text is easy to read with good contrast Buttons are clearly labelled for easy navigation

The layout ensures that users can interact with the app without confusion.

🧪 Testing

The app was tested using an Android emulator.

The following tests were performed:

Valid inputs (morning, afternoon, night) display correct results Invalid inputs show an error message Empty input is handled properly Reset button clears all fields

Logging was also used in the code to check user input during testing.
